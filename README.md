# Travel Agent

Zero-cost travel planning web app.

## Stack
- Frontend: React + Vite
- Backend: FastAPI
- LLM: Groq

## Layout
- `frontend/` React app
- `backend/` FastAPI app

## Environment
- Copy `.env.example` to `.env` and fill in your keys.
- Required now: `GROQ_API_KEY`

## Next steps
1. Install frontend and backend dependencies.
2. Run the backend API.
3. Run the React dev server.
